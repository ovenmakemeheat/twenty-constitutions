# X.X การเผยแพร่และการเข้าถึงข้อมูล (Data Publishing & Public Access)

หลังจากผ่านกระบวนการเตรียมข้อมูลและวิเคราะห์เชิงสำรวจแล้ว ผู้จัดทำได้เผยแพร่ผลลัพธ์ในสองรูปแบบหลัก ได้แก่ (1) ชุดข้อมูลสาธารณะบนแพลตฟอร์ม Kaggle เพื่อรองรับการนำไปใช้งานต่อในเชิงวิจัย และ (2) เว็บแอปพลิเคชัน Dashboard แบบโต้ตอบสำหรับสาธารณะทั่วไป โดยมีซอร์สโค้ดเปิดเผยบน GitHub

---

## X.X.1 ชุดข้อมูลสาธารณะบน Kaggle

ผู้จัดทำได้เผยแพร่ชุดข้อมูลภายใต้ชื่อ **"Twenty Constitutions Digitization"** บนแพลตฟอร์ม Kaggle ซึ่งเป็นชุมชนนักวิทยาศาสตร์ข้อมูลและนักวิจัยที่ใหญ่ที่สุดแห่งหนึ่งของโลก ชุดข้อมูลประกอบด้วยไฟล์หลักสองไฟล์ ได้แก่ `preprocessed_data.csv` (2,797 แถว 21 คอลัมน์ ครอบคลุมรัฐธรรมนูญ 38 ฉบับ ตั้งแต่ พ.ศ. 2475–2564) และ `section_theme_scores.csv` (5,188 แถว ผลการจัดหมวดหมู่เชิงหัวข้อ 13 ด้าน) เพื่อให้นักวิจัย นักศึกษา และผู้สนใจสามารถดาวน์โหลดและนำไปวิเคราะห์ต่อได้อย่างอิสระ

![หน้า Dataset Card ของ Twenty Constitutions Digitization บน Kaggle แสดงชื่อชุดข้อมูล คำอธิบาย จำนวนไฟล์ ขนาด และ Metadata ทางสถิติ](images/kaggle_dataset_card.png)
*รูปที่ X.X หน้า Dataset Card ของชุดข้อมูล "Twenty Constitutions Digitization" บน Kaggle แสดงรายละเอียดชุดข้อมูล ขนาดไฟล์ และสถิติภาพรวม*

ชุดข้อมูลดังกล่าวสามารถเข้าถึงได้ที่:
**https://www.kaggle.com/datasets/microhum/twenty-constitutions-digitization/**

โครงสร้างของ Dataset Card ที่เผยแพร่ประกอบด้วย

- **ชื่อและคำอธิบายชุดข้อมูล** อธิบายบริบทโครงการ ขอบเขตเวลา และวิธีการเตรียมข้อมูล
- **ไฟล์ข้อมูลหลัก** `preprocessed_data.csv` และ `section_theme_scores.csv` พร้อม Data Dictionary ครบถ้วน
- **Metadata ทางสถิติ** ได้แก่ จำนวนแถว คอลัมน์ ขนาดไฟล์ และการกระจายตัวของตัวแปรหลัก
- **Usability Score** และ License ที่กำหนดเงื่อนไขการใช้งาน

การเผยแพร่บน Kaggle ช่วยให้ชุดข้อมูลเข้าถึงได้ผ่าน Kaggle Notebooks โดยตรงโดยไม่ต้องดาวน์โหลด รวมถึงรองรับการนำไปใช้ใน Machine Learning Pipeline ผ่าน Kaggle API

## X.X.2 เว็บแอปพลิเคชัน Dashboard (Twenty Constitutions Web)

นอกเหนือจากชุดข้อมูลดิบ ผู้จัดทำได้พัฒนาเว็บแอปพลิเคชันแบบโต้ตอบสำหรับเผยแพร่ข้อมูลในรูปแบบที่เข้าถึงได้ง่ายสำหรับสาธารณะทั่วไป โดยไม่จำเป็นต้องมีความรู้ด้านการวิเคราะห์ข้อมูล

ซอร์สโค้ดของโครงการเปิดเผยบน GitHub ที่:
**https://github.com/beambeambeam/twenty-constitutions**

แอปพลิเคชันพัฒนาด้วย React Router v7 (SSR) บน TypeScript พร้อม Tailwind CSS v4 โดยมีฟีเจอร์หลักดังนี้

**หน้าภาพรวม (Overview Dashboard)**

![หน้าแรกของ Twenty Constitutions Dashboard แสดง Constitutional Timeline แบบ Gantt Chart ที่แสดงรัฐธรรมนูญ 38 ฉบับตามแนวนอน ความสูงของแถบแสดงจำนวนมาตรา สีแสดงประเภทระบอบ พร้อม Heatmap ของน้ำหนักเชิงหัวข้อ 13 ด้านแยกตาม 8 ยุคสมัย และตารางรัฐธรรมนูญทั้งหมดพร้อมฟิลเตอร์](images/web_overview_dashboard.png)
*รูปที่ X.X หน้า Overview Dashboard แสดง Constitutional Timeline, Theme × Era Heatmap, สถิติภาพรวม และตารางรัฐธรรมนูญทั้งหมดพร้อมระบบกรองข้อมูล*

หน้า Overview นำเสนอข้อมูลสำคัญครบในหน้าเดียว ประกอบด้วย

- **Stat Bar** แสดงตัวเลขสรุป: 38 ฉบับ, 2,797 มาตรา, 89 ปี, 13 หัวข้อ
- **Constitutional Timeline** แผนภูมิ Gantt แบบโต้ตอบ โดยแต่ละแถวคือรัฐธรรมนูญหนึ่งฉบับ ความกว้างของแถบแทนช่วงเวลาที่มีผลบังคับใช้ ความสูงแสดงจำนวนมาตรา และสีแสดงประเภทระบอบการปกครอง (Civilian/Military/Semi-Military) เมื่อเลื่อนเมาส์จะแสดง Animated Tooltip แบบ `|_` ที่แสดงข้อมูลเชิงลึก เมื่อคลิกจะเปิด Drawer รายละเอียด
- **Theme × Era Heatmap** ตาราง 13 หัวข้อ × 8 ยุคสมัย ใช้ความเข้มของสีแทนน้ำหนักเชิงหัวข้อ คลิกที่แถวใดแถวหนึ่งเพื่อกรองตารางด้านล่างให้แสดงเฉพาะยุคนั้น
- **ตารางรัฐธรรมนูญ** พร้อมระบบค้นหา กรองตามระบอบ ประเภทเอกสาร และยุคสมัย รองรับการเรียงลำดับแบบ Multi-column

**Document Drawer**

เมื่อผู้ใช้คลิกที่รัฐธรรมนูญในตาราง หรือในแผนภูมิ Timeline แอปจะเปิด Slide-in Drawer จากด้านขวาแสดงรายละเอียดของเอกสารนั้น ประกอบด้วย

- หัวข้อเอกสาร ปีพุทธศักราช ประเภท ระบอบ และยุคสมัย
- แผนภูมิแท่ง Theme Profile แสดงน้ำหนักเชิงหัวข้อ 13 ด้านของเอกสารนั้น
- แผงนำทางบท (Chapter Navigation) ด้านซ้ายสำหรับกรองเนื้อหาตามบท
- รายการมาตราทั้งหมด พร้อม Theme Pills และระบบขยาย/ย่อข้อความมาตรา

แอปพลิเคชันนี้เผยแพร่บน Cloudflare Pages ที่:
